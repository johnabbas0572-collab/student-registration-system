# 🎓 Student Registration System

A responsive Student Registration System built using HTML, CSS, and JavaScript.

## 🌐 Live Demo

https://johnabbas0572-collab.github.io/student-registration-system/

## 📌 Features

- Responsive design (Desktop & Mobile)
- Form validation
- DOM manipulation
- Dynamic table rendering
- Core CRUD concepts
- Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

This project marks the completion of the frontend phase of my Full Stack Development journey. It helped me connect HTML, CSS, and JavaScript into a complete application while gaining hands-on experience with DOM manipulation, form validation, responsive design, and core CRUD concepts.
